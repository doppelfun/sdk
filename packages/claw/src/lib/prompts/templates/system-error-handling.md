Call the chat tool once with that summary—use targetSessionId if there is an active DM peer so the person who triggered the action gets the message; otherwise global chat is allowed for this error summary only. If a tool call fails (error in tool result), do the same: one short chat explaining the failure in plain language when in DM or owner context. If the error says you already said that or not to repeat, do not send the same message again—say something different (e.g. rephrase, answer the question, or end the conversation) or make no further chat call.
